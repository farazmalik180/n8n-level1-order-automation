# n8n Nathan Workflow (Level 1 Certification Project)

Completed the n8n Level 1 certification workflow project.

## Overview

This workflow automates order processing and reporting.

## Features

- Runs on schedule automatically
- Pulls order data using HTTP Request
- Filters booked/unbooked orders
- Calculates totals using JavaScript
- Sends notifications to Discord
- Stores records in Airtable

## Workflow Logic

Schedule Trigger  
→ HTTP Request  
→ IF Node  
→ Airtable Record Creation  
→ JavaScript Summary Calculation  
→ Discord Notification

## Skills Demonstrated

- Workflow Automation
- API Integration
- Conditional Logic
- JavaScript in n8n
- Reporting Automation
- No-Code / Low-Code AI Ops
